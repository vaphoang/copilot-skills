# copilot-skills
A repo containing useful Copilot skills.

## Skills
- `squash-branch-commits`: Squash branch commits into one commit safely.
- `pr-address-comments`: `gh`-based workflow to review PR feedback, let the user select comments to address (preferably via checkbox multi-select), then reply to and resolve fixed bot threads.
- `pr-create`: Create a new pull request in a GitHub repository with `gh`, gather details, show a summary, and ask for confirmation before creating.
