# Suggested commands
- List files: `ls`, `find . -maxdepth 3 -type f`
- Inspect repo status: `git status --short`, `git diff --stat`
- Search text: `grep -R "pattern" skills README.md`
- Read markdown: `cat README.md`, `cat skills/<skill>/SKILL.md`
- GitHub CLI check: `command -v gh >/dev/null && gh --version`
- Preview/install skills (if published via gh skill features): `gh skill preview <repo> <skill>`
- Useful macOS shell commands: `pwd`, `cd`, `grep`, `find`, `sed`, `awk`, `bash -n <script>`.
